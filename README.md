# DominoTiling

На доске M×N отмечены некоторые клетки. Требуется покрыть отмеченные клетки
доминошками 1×2 стоимостью А рублей и квадратиками 1×1 стоимостью В рублей,
потратив минимальное количество денег.

Формат ввода:
В первой строке написаны числа N, M, A, B. Затем следуют N строк по M символов. 
Символ «.» обозначает неотмеченную клетку, а символ «*» --- отмеченную.

Формат вывода:
Вывести одно число --- минимальную сумму денег.
